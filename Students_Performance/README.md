This folder contains the Students Performance dataset along with the exported Kibana dashboard. The dashboard analyzes student performance across multiple factors such as gender, parental education, lunch type, and test preparation course. The goal is to visualize patterns and gain insights into academic performance.
# Student Performance Dashboard (Kibana)

This folder contains:
- Student performance dataset (CSV)
- Kibana dashboard export file
- Visualizations created using Lens

## Files
- students_performance.csv – original dataset
- students_dashboard.ndjson – exported Kibana dashboard & visualizations

## How to use
1. Import the CSV into Elasticsearch
2. Open Kibana → Stack Management → Saved Objects → Import
3. Upload students_dashboard.ndjson
4. Dashboard will be recreated
